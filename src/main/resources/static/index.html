ℹ️ How Auto-Split Works
When you add an expense, the backend automatically divides the total amount equally among all members of the group. Each member's share is tracked as a Split record with status UNPAID.
The person who paid is excluded from owing themselves. Others can settle their splits via the Transactions → Settle flow using the split ID returned in the expense response.
